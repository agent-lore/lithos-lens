# Lithos Lens User Manual

Lithos Lens is a read-only window onto **Lithos** coordination state. Use it to watch the tasks agents are working on, drill into a single task, follow the findings that agents post against it, and read the knowledge notes those findings link to. Lens never changes anything in Lithos — it only shows you what is there, live.
